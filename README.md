# 🌐 Network Security & Basic Traffic Monitoring – Week 2 Task 2

## Overview

This repository covers the fundamentals of network security, common network-based attacks, and basic traffic analysis using Wireshark. Created as part of **Week 2 – Task 2** of the Cybersecurity Internship program.

## Learning Objectives

- Understand how data travels across networks (IP, DNS, HTTP, HTTPS)
- Identify common network threats (MITM, DDoS, Packet Sniffing)
- Perform safe, legal traffic observation using Wireshark
- Understand how attackers exploit network weaknesses
- Apply network security best practices
